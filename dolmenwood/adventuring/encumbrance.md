# 负重

> 规定冒险者可以携带多少用品和财宝。

在凶险荒野之中，冒险者的装备生死攸关。一根绳索可以防止坠入死亡的深渊，一柄燃烧的火炬可以抵御野兽的袭击，一根不起眼的钓竿可以填饱饥饿的肚子。角色在冒险时必须审慎考虑随身携带的装备，因为每人只能携带这么多。所携带的每件物品既是负担，也可能是危急时刻的救命稻草。

这里介绍两种确定角色可携带物品总量的方法：一是记录携带物品的总重量，二种是记录携带物品的数目。裁判应决定使用哪种方法，并应用于所有角色。

## 重量负重

此负重系统基于记录所携带物品个体重量，可以详尽并相当真实地了解每名角色的负载情况。

### 载重量

应记录角色用品的总重量。由于钱币是冒险者最常见的财宝形式，所有重量都以钱币为单位。十钱币等于一磅。

**装备：**装备列表记录了所有武器、护甲和冒险用品的重量。希望简化记录的团体可将角色的杂项用品（譬如背包、绳索、火炬等）——不包括武器与护甲——的总重量视为 100 钱币。

**财宝：**应记录角色所携带财宝的重量。见***常见财宝重量***。

**最大载重：**任何角色的最大载重为 1600 钱币。携带更多的角色无法移动。

### 确定速度

角色速度取决于其携带物总重量。见***重量负重***表。

### 重量负重

| **总重量**     | **速度** |
| :------------- | :------- |
| 至多 400 钱币  | 40       |
| 至多 600 钱币  | 30       |
| 至多 800 钱币  | 20       |
| 至多 1600 钱币 | 10       |

### 马匹与载具

坐骑与载具的载重量以钱币形式列于***马匹与载具，p120***。

## 基础重量负重（可选规则）

若团体偏好更简易的重量负重系统，可选用此选项。

**装备：**不记录护甲、武器和冒险用品的重量，也不将其计入角色的最大载重。

**财宝：**应记录所携带财宝的重量，以确保其不超出角色的最大载重。见***常见财宝重量***。

**最大载重：**任何角色的财宝最大载重为 1600 钱币。携带更多的角色无法移动。

### 确定速度

角色速度取决于所穿护甲类型以及是否携带大量财宝（裁判裁定）。见***基础负重***表。

### 基础负重

| **护甲笨重** | **无财宝速度** | **有财宝速度** |
| :----------- | :------------- | :------------- |
| 无           | 40             | 30             |
| 轻甲         | 30             | 20             |
| 中甲或重甲   | 20             | 10             |

## 常见财宝重量

常见财宝的重量情况如下。裁判应决定其他形式财宝的重量。

### 财宝重量

| **物品**         | **重量** |
| :--------------- | :------- |
| 钱币（任何种类） | 1        |
| 宝石             | 1        |
| 珠宝（1 件）     | 10       |
| 药剂             | 10       |
| 权杖             | 20       |
| 卷轴             | 1        |
| 杖               | 40       |
| 魔杖             | 10       |

## 栏位负重

此负重系统基于*用品*栏位，简易抽象地概括了每名角色的负重。每件所携带物品占用 0–3 个用品栏位——见***物品占用栏位***。取决于所占用品栏位，角色会变得缓慢沉重。

### 配备物品：10 栏位

角色拥有 10 个*配备物品*栏位，即角色身上穿着、手持、正在使用或预备就绪的任何物品：穿着的护甲、手持的盾牌或武器、佩鞘的武器、腰包中的物品。

**最大载重：**角色不可携带超过 10 个栏位的配备物品。

### 存放物品：10 栏位/容器

麻袋和背包各有 10 个*存放物品*栏位，即存入容器中的物品。

**最大载重：**角色不可携带超过 16 个栏位的存放物品。

**最多容器：**角色可佩戴一个背包，可携带至多两个麻袋且各占一只手。角色手持麻袋时，占用手不可携带武器、盾牌、提灯或其他物品。

**取回：**在战斗中，取回存放物品花费 1 轮。

### 确定速度

角色速度取决于其所占用品栏位，如***栏位负重***表所示。应分别记录配备物品和存放物品的数量，在表中查找两者并采用所列的最慢速度。

### 栏位负重

| **占用栏位** |          |          |
| :----------- | :------- | :------- |
| **配备**     | **存放** | **速度** |
| 0–3          | 0–10     | 40       |
| 4–5          | 11–12    | 30       |
| 6–7          | 13–14    | 20       |
| 8–10         | 15–16    | 10       |

### 马匹与载具

坐骑与载具的物品栏位数目以其钱币载重除以 100 得出（见***马匹与载具，p120***）。配备物品与存放物品之间的区别不适用于马匹与载具。

## 物品占用栏位

**所占用品栏位**表列出了各种物品会占用的用品栏位。

**通用物品：**除非另有注明，否则角色所携带的每件物品都占用 1 个用品栏位。

**微小物品：**一些很小物品仅在大量携带时（裁判裁定）才会占用 1 个用品栏位。

**叠放物品：**有些物品可打包购买并叠放携带。一套占用 1 用品栏位。

**容器：**容器（譬如背包、麻袋）仅在不使用时才算作一件物品。

**笨重物品：**需双手搬运的大型或重型物品占用 2 个用品栏位，且无法存放。裁判裁定，特重或笨重的物品可占用 3 个以上用品栏位。

**服装与护甲：**服装不占用物品栏位。轻甲占用 1 个栏位，中甲占用 2 个栏位，重甲占用 3 个栏位。盾牌占用 1 个栏位。

### 所占用品栏位

| **冒险用品**                                                 | **栏位**   |
| :----------------------------------------------------------- | :--------- |
| 通用物品                                                     | 1          |
| 衣物                                                         | 0          |

| 叠放物品：譬如蜡烛（10）、火炬（3）、铁蒺藜（20）、粉笔（10）、铁钉（12弹珠（20） | 1/叠       |
| 容器：譬如背包、腰包、麻袋                                   | 使用时为 0 |
| 笨重物品：譬如桶、匣子、箱子、杆子（长 10'）                 | 2          |
| 钱币或宝石（至多 100）                                       | 1          |

| **武器**                 | **栏位** |
| :----------------------- | :------- |
| 近战武器，单手           | 1        |

## 丢弃物品

当角色发现大量宝藏时，也许会不得不丢下物品，舍弃冒险用品，以便把金子塞满口袋。

在危急关头，譬如在逃离敌人时，角色可选择丢弃物品来减轻负担。为了增加角色速度，武器、盾牌、甚至是满载的麻袋或背包都可丢弃。
